üêõ COMMON ISSUES & SOLUTIONS
================================================================================

Issue: "Please add at least one expense"
Solution: Click "+ Add MP/BERHAD Expense" or "+ Add Market Expense"

Issue: "Missing Receipt/Voucher Upload"
Solution:
   1. Expand the expense card
   2. Click on the red upload field
   3. Select JPG/PNG/PDF file
   4. Verify field turns GREEN

Issue: Button does nothing
Solution:
   1. Open browser console (F12)
   2. Click submit button
   3. Check console for error messages
   4. Verify all required fields are filled

Issue: File won't upload
Solution:
   1. Check file is JPG, PNG, or PDF
   2. Check file is less than 5MB
   3. Try a different file
